RLink

RLink is an advanced VB.NET Plugin that works with the popular Analyst’s Notebook Versions including 6.1.

While I2’s IBridge is Read-Only RLink brings Write-Back & Write-Thru to Standard Analyst’s Notebook Installations. It features GUI-controlled Mapping of Schema and  XML-based Configuration Data
